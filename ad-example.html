üìù How to Use

Method 1: Auto-init (Easiest)

<!-- Include the script -->
<script src="blessedbump-ad.js"></script>

<!-- Add container with auto-init -->
<div id="blessedbump-ad" data-auto-init="true"></div>

Method 2: Manual Init

<!-- Include the script -->
<script src="blessedbump-ad.js"></script>

<!-- Add container -->
<div id="my-ad-container"></div>

<!-- Initialize -->
<script>
  BlessedBumpAd.init({
    container: 'my-ad-container',
    websiteUrl: 'https://blessedbump.in',
    tagline: 'Your Journey, Your Glow, Your Village',
    trackClicks: true,
    trackViews: true
  });
</script>

Method 3: Custom Configuration

BlessedBumpAd.init({
  container: 'blessedbump-ad',
  websiteUrl: 'https://blessedbump.in',
  tagline: 'Your Journey, Your Glow, Your Village',
  primaryColor: '#FF9FB8',
  secondaryColor: '#7C72FF',
  minHeight: '200px',
  trackClicks: true,
  trackViews: true,
  animation: true,
  analyticsEndpoint: 'https://your-analytics-endpoint.com/track'
});

‚öôÔ∏è Configuration Options

  • container - ID of the container element (default: 'blessedbump-ad')
  • websiteUrl - Your website URL (default: 'https://blessedbump.in')
  • tagline - Ad tagline (default: 'Your Journey, Your Glow, Your Village')
  • primaryColor - Primary brand color (default: '#FF9FB8')
  • secondaryColor - Secondary brand color (default: '#7C72FF')
  • minHeight - Minimum ad height (default: '200px')
  • trackClicks - Track click events (default: true)
  • trackViews - Track view events (default: true)
  • animation - Enable animations (default: true)
  • analyticsEndpoint - Custom analytics endpoint URL (optional)

üìä Analytics

The script automatically tracks:

  • Views - When the ad is displayed
  • Clicks - When users click the ad

Events are logged to console and can be sent to:

  • Google Analytics (if gtag is available)
  • Custom analytics endpoint (if configured)
