### Step 1: Go to APIs & Services Library

1. **Sign in to Google Cloud Console**

   - Look at the top of the page

### Step 2: Search for Google+ API

Once you're on the Library page:

1. **Find the Search Box**
   - At the top of the page, you'll see a search box
   - It might say "Search for APIs & Services" or just have a magnifying glass icon

2. **Type in the Search Box**
   - Click on the search box
   - Type: `Google+ API`
   - OR type: `Google Identity Services API` (newer version)

3. **Look for Results**
   - You should see search results appear below
   - Look for either:
     - **"Google+ API"** (older, but still works)
     - **"Google Identity Services API"** (newer version)
   - Either one will work!

---

### Step 3: Click on the API

1. **Click on the API Name**
   - Click on either:
     - **"Google+ API"** (if it appears)
     - OR **"Google Identity Services API"** (preferred, newer)
   - This will open the API details page

---

### Step 4: Enable the API

1. **Find the Enable Button**
   - On the API details page, you'll see a big blue button
   - It will say **"Enable"** or **"ENABLE"**

2. **Click Enable**
   - Click the blue **"Enable"** button
   - Google will start enabling the API

3. **Wait for Confirmation**
   - You'll see a loading spinner or progress message
   - Wait a few seconds (usually 5-10 seconds)
   - Once enabled, you'll see:
     - A checkmark âœ…
     - Message like "API enabled" or "Google+ API has been enabled"
     - The button might change to "Manage" or "Disable"
